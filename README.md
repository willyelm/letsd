# Letsd –– simple task interface builder.

> Work In Progress

Letsd makes it easy to create interactive command line tools

## Installation

To install letsd using `npm`:

```bash
npm install letsd
```

or using cURL:

```bash
curl -o- https://raw.githubusercontent.com/willyelm/letsd/v0.0.1/install.sh | bash
```
